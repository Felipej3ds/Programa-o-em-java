# Conteudo
 Os estudos são voltado a formação de um profissional na área de software

- Fundamentos:
	 então primeiramente focaremos em aprender conceitos básicos essenciais para qualquer desenvolvedor(Programação orientada a objetos e estrutura de dados), então após isso aprenderemos algumas ferramentas básicas para qualquer desenvolvedor como (Banco de dados, SQL e Git) e também trata sobre analise de design de sistemas  	 
- Comercial:
	- Após estabelecer fundamentos para programação o curso passará para a parte comercial que consiste no ensinamento dos ferramentas importantes relacionadas a linguagem para o desenvolvimento de aplicativos Java
		- Acesso à banco de dados via JDBC
			- Aprenderemos fundamentos
			- aprenderemos a trabalhar com o padrão de projetos de acesso a dados
				- como fazer uma consulta SQL para trazer os dados na forma de tabela e converter esses dados na forma de objeto de memoria
